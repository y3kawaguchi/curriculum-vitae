## 主な経歴

### スマホ決済におけるAML/CFTシステム開発【Go/React/AWS/GCP】(2019年10月〜現在)

【プロジェクト概要】

スマホ決済サービスから、マネーロンダリングやテロ資金供与が疑われる決済をモニタリングするシステムを開発。システムはマイクロサービスアーキテクチャを採用しており、大きく、決済受信サービス、疑わしい決済検知サービス、オペレータが利用するモニタリングツールの3つに分かれる。
